工作內容介紹

實習期間完成進度

學習 - 運用Java + SQL + HTML + JavaScript + JQuery 寫系統

自我評估及心得感想

        從實習開始到現在已經過了三個多月，過去的一百多天裡，除了實習還有上課，感覺生活不斷被這兩件事情重複追著跑，難得放假的星期日只想躺在床上什麼事情都不想做。雖然很累，卻過得很充實。從一開始會擔心自己需要多久的時間才能適應一個新的環境，也會擔心自己能力的不足。到現在已經習慣這樣的生活，八點打卡上班，五點不一定準時下班。在公司工作真的很不一樣，短短的三個月，程式設計的能力卻好像比學校一年所學的還多，每天都有不一樣的任務，每天都學到不一樣的方法去解決使用者的問題，上班的每一天都很期待中午休息時間…

        現在的我，只是個小小實習生，能力有限，像個海綿一樣，每天吸收不同的東西，不斷的膨脹。工作三個月，請實習公司的主管們填寫問卷，藉著這個機會，看看他們對我的評價是什麼，有哪裡需要改進的地方，接下來還有四分之三的合約，而未來是未知數。

        時間過的真的很快，不知不覺來到技嘉已經七個多月了，對於一邊上課一邊實習的過程，已無過多的感覺。聽到同學們在不同公司上班，而有不同的心得，反而還覺得有趣，也深深地感受到大家的成長。從前不曾有交集的同學，也因為在同一間公司上班，才有機會更認識他們。

        到目前為止，幾乎所有的工作內容都是在更新程式或是建立新系統。而系統主要的功能都是使用者輸入條件後，傳回資料並顯示於頁面上。雖然工作內容好像都相同，但是根據不同使用者的需求，程式都會附加不一樣的功能。因此，即使每次的工作項目都是查詢，然後顯示結果，仍然會有不懂的地方需要學習。畢竟，學校所學只是入門。而那些知識是學不完的，正所謂學海無涯，尤其在現今這資訊流通如此快速的時代，加上我所在的行業正好是資訊產業，每天都有功能推陳出新。如果不能讓自己跟上這潮流，是很容易被社會淘汰的。

        以往，我的工作項目都是主管和使用者討論過系統的需求後，才分配工作內容給我。而前些時候，帶領我的工程師因家中有事，請假了一陣子。因此，當使用者發現程式有什麼需要修改的地方時，就會直接寫E-mail給我，這是以往不曾有過的。

        實際來到公司上班，才能體會賺錢的辛苦，以及職場上那些不能說，大家卻都照做的淺規則，這也算是另外一種收穫吧，我想。
