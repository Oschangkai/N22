學習

測試種類

黑箱測試是直接就功能面來驗證，而白箱測試是以程式語言的角度來測試，這次實習所學習的主要為黑箱測試的部分

CGI

CGI是提供瀏覽器和 server 程式溝通的方式，在實習中，我學習到如何下CGI指令，來查詢和更改參數

Camera and NVR

以前從未接觸過這類型產品，進了晶睿後才學習到，攝影機與網路錄相機的使用與架設方式                             

溝通與表達

處理問題的過程中學習到，要如何簡潔且明確的描述出問題的情況給他人知道，並且要相互達成協議，問題交由誰處理，這樣才是一個完美的溝通

網路知識、網路架構

在架設有線與無線的Camera和NVR期間，接觸到很多的網路知識與架構，實際操作後才發現，網路世界相當複雜，很多原因都會阻礙架設

Python與爬蟲

為維護公司內部的版號網站，而接觸到 Python 與爬蟲程式，透過主管的教學和網路查資料自學，在實際操作和應用後，學習到許多

心得感想

經過整個學年和暑期的歷練，對於測試韌體的工作越來越得心應手，遇到問題時的處理方式、溝統和表達的能力都有微微的進步，且跟同事間的相處也更加地融洽。在最近幾個月裡，深刻的感覺到什麼是「公司的風格和主管管理的風格」，對於我而言，是第一次進入企業工作，因此並沒有辦法與過去的經驗做出必較，但在未來要選擇公司時，選擇一間喜歡的公司，公司的風格是相當重要的，而另外主管管理的風格，在我的實習期間，剛好遇到了小組換組長，對我來說新舊組長都很棒，對每個組員都很用心，不過管理的方式卻很不一樣，在我們部門中，不同主管有很多不同的管理風格，當然每種管理風格有皆他的好與壞，只要自己能接受就是一種好得管理風格。
